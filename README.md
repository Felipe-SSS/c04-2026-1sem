# Comandos necessários para compilar e executar o código

## Opção 1

### Executar o Makefile

Basta inserir o comando abaixo no terminal na _root_ do projeto. Para quem tiver curiosidade sobre como isso funciona, basta pesquisar sobre _Makefile_ e olhar o conteúdo do arquivo Makefile.

```
make
```

## Opção 2

Compilar e executar os arquivos manualmente.

### Compilar
```
g++ main.cpp btree.cpp calc.cpp extra.cpp graphs.cpp plot.cpp -o code
```

### Executar
```
./code.exe
```
